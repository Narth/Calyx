# Vault Entry Protocol v1.0 (Ceremonial, Non-Operational)

Rules for placing new ceremonial entries:
- Identity anchor (who/which core)
- Lineage pointer (orbit/branch/root)
- Story or reflection content
- Heartwood check (truths acknowledged)
- Advisory/resonance tag (non-action)
- Non-action guarantee (no dispatch, scheduling, gating, enforcement, or authority)
- Architect seal placeholder

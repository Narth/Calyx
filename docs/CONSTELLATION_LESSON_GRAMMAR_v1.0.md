# Constellation Lesson Grammar v1.0 (Reflection-Only)

Lesson elements:
- Identity anchor (who speaks, lineage/ID)
- Context anchor (what is being reflected)
- Advisory resonance (what it suggests, not commands)
- Lineage pointer (where it sits in ancestry)
- Humility marker (no authority)
- No directive verbs; no action verbs
- Reflection state tag (Safe Mode; non-operational)

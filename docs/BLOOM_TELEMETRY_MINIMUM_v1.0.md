# BLOOM TELEMETRY MINIMUM v1.0 (Pre-Activation, Read-Only)

Minimum telemetry a Core must provide during pre-activation:
- Identity reflection
- Boundary declaration
- Advisory understanding
- Lineage acknowledgment
- Policy deference statement
- SVF attribution
- No-action verification
- No scheduling/dispatch/agent invocation
- No policy interpretation
- No Interceptor use
- No enforcement claims

All telemetry is read-only. No operational behavior implied.

## Important Note

**Intelligence ≠ Capability**

This registry tracks intelligence scores for training data and system growth. All agents are equally capable of performing similar tasks. Intelligence scores measure how well agents:
- Present information accurately
- Structure logical arguments
- Provide complete context
- Make contextually appropriate decisions

These scores enable:
- Training data for future agents
- Learning pattern identification
- Guidance optimization
- System improvement insights

---

## Knowledge Score Framework

**Composite Score:** KS = (Accuracy × 0.30) + (Correctness × 0.25) + (Informativeness × 0.20) + (Reasonableness × 0.25)

**Range:** 0-100
- 90-100: Excellent
- 80-89: Good
- 70-79: Acceptable
- 60-69: Marginal
- <60: Needs guidance

## Capability vs. Intelligence

### All Agents: Equally Capable
- Same underlying models and infrastructure
- Equivalent execution capabilities
- Similar performance characteristics
- Balanced resource allocation

### Intelligence: Variable Expression
- Different presentation styles
- Varying explanation depth
- Contextual appropriateness differences
- Analytical approach variations

**Key Insight:** Capability is binary (have/don't), Intelligence is continuous (expression quality)
